An interactive, premium-grade developer portfolio built using React, Vite, and custom CSS. This application is designed as a futuristic "Command Center" dashboard displaying real-time developer metrics, project showcases, interactive charts, and system status widgets.

---

## 🌟 Key Features

*   **Interactive Particle Background**: Ambient background powered by a dynamic HTML5 Canvas particle system.
*   **Holographic Profile Card**: Orbiting electron animations with live status metrics (Projects, Certifications, and LeetCode problem counts).
*   **Live Metrics Counters**: Dynamic counter statistics with custom micro-animations showing achievements.
*   **Dynamic Skill Matrix**: An orbiting 3D skill-web of technologies centering on the core expertise, supplemented by responsive progress bars.
*   **Interactive Project Showcases**: Expandable modal overlays presenting technology, features, and source links for core AI & ML projects.
*   **LeetCode & GitHub Intelligence**: Live chart analysis visualization and heatmaps representing coding consistency.
*   **AI Assistant Integration**: An interactive AI chat window layout letting visitors query interests, background, and projects directly.

---

## 🛠️ Technology Stack

*   **Framework**: [React](https://react.dev/) (v18)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Styling**: Vanilla CSS with futuristic glassmorphism, responsive grid layouts, custom keyframes, and scanline/glow effects.
*   **Components**: Custom React components mapping the system stats, timeline, and interactive modal transitions.

## 🚀 Getting Started

### Prerequisites
Make sure you have **Node.js** installed on your system.

### Installation & Run

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/kicha7887/kishore-portfolio.git
    cd kishore-portfolio
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Your application will be live at `http://localhost:5173`.

4.  **Build Production Package**
    ```bash
    npm run build
    ```
